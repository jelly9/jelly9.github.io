如何让你的简历脱颖而出

发表于 2017-03-03 | | 阅读次数

· 个人信息

·姓名
·电话:xxx-xxxx-xxxx（344格式顺口）
·邮箱:
·毕业院校:
·所学专业:
·学历:

· 求职信息

·清晰明了
·有针对性
·纸质简历（可修改）

· 专业技能

·真实呈现（不夸大）
·条理清晰（有逻辑）
·完全展示自己（不要害羞）

· 特殊技能

·有扎实的编程基础
·有宽阔的视野（svn、git、github、stackoverflow、开源项目涉猎、热门技术的了解）
·学习精神、吃苦精神（潜在的能力）。

· 项目经验

·真实（自己做过、并熟悉）
·有趣（让面试官感兴趣）
·有料（技术特点的展示）

· 简历中的措辞

·项目措辞：以动词为主，强调我做了什么，实现了什么！

· 简历投递注意事项

·避免使用qq邮箱，可以使用高大上的，比如谷歌雅虎。。。
·发件人名称（可修改）
·清晰的邮件主题（如: 姓名-岗位名称）
`不要发群
·不要使用压缩文件
·不要使用云下载
·文件格式尽量是pdf，word也行，还有排版等等，要看起来顺眼。
·突出重点
·字体协调（微软雅黑、Times、NerRoman）
·注重细节（错别字、专业用词、大小写）
·简历模板很重要（乔布斯简历 or 自己设计简历）
